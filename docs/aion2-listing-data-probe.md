# 永恒之塔2售卖数据单页探针规范

## 目标

验证螃蟹游戏服务网的永恒之塔2台服账号列表页中，已经渲染的公开售卖条目可以被程序化读取。

目标页面：

```text
https://www.pxb7.com/buy/175178554941486/1
```

## 第一阶段范围

- 在用户正常打开的目标列表页中读取 DOM。
- 按站点单次列表大小最多返回当前页面前 16 条商品。
- 输出字段为区名、种族、价格、职业、装等、战斗力、发布时间和详情链接。
- 输出 JSON 到浏览器控制台并返回同一对象，便于后续接入自动化执行器。
- 不保存完整页面，不写数据库，不建立定时任务。

## 明确禁止

- 不请求下一页，不触发无限滚动，不遍历商品详情。
- 不直接调用站点内部接口，不复用或导出浏览器 Cookie、令牌和指纹数据。
- 不规避验证码、JavaScript 挑战、WAF、限流或登录要求。
- 不采集卖家身份、联系方式、支付信息或其他非公开个人数据。

## 运行与验证

浏览器探针位于：

```text
scripts/probe-aion2-listings.browser.mjs
```

静态与行为边界校验：

```bash
node scripts/validate-aion2-data-probe.mjs
```

完整仓库校验：

```bash
bash init.sh
```

## 进入下一阶段前

需要先确认站点授权或服务条款、允许的请求频率、目标字段、去重策略、保存周期和删除机制。未经确认，不把本探针扩展为网络采集器。
